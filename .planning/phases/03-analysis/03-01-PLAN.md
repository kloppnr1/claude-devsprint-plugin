1. `parseVstfsRefUri(vstfsUrl)` — parses `vstfs:///Git/Ref/{projectId}/{repositoryId}/GB{branchName}` URIs.
   - Strip `vstfs:///Git/Ref/` prefix, split on `/`
   - parts[0] = projectId, parts[1] = repositoryId
   - parts[2..] joined with `/`, then strip `GB` prefix = branchName
   - Return `{ projectId, repositoryId, branchName }`
   - Handle branch names with slashes (e.g., `feature/US-1234-add-checkout`)

2. `resolveRepository(org, project, repositoryId, encodedPat)` — looks up a git repository by ID.
   - First try project-scoped: `GET {org}/{project}/_apis/git/repositories/{repositoryId}?api-version=7.1`
   - If 404, fall back to org-level: `GET {org}/_apis/git/repositories/{repositoryId}?api-version=7.1` (handles cross-project repos per Pitfall 4 in RESEARCH.md)
   - Return parsed JSON `{ id, name, remoteUrl, defaultBranch }` on success
   - Throw Error if both return non-200

Add `cmdGetBranchLinks(cwd, args)` async function and wire it into the CLI router.

**Command contract:**
```
node azdo-tools.cjs get-branch-links --id <workItemId> [--cwd <path>]
  stdout: JSON array [{ repositoryId, repositoryName, remoteUrl, branchName }]
  Returns [] if no branch link found (not an error)
  exit 0 on success, exit 1 on API error
```

**Implementation:**
1. Parse `--id` from args. If missing, stderr "Missing --id" and exit 1.
2. `loadConfig(cwd)`, encode PAT with `Buffer.from(':' + cfg.pat).toString('base64')`
3. Fetch work item with relations: `GET {org}/{project}/_apis/wit/workitems/{id}?$expand=relations&api-version=7.1`
4. If non-200, throw error with HTTP status.
5. Filter relations: `r.rel === 'ArtifactLink' && r.url && r.url.startsWith('vstfs:///Git/Ref/')` — use URL-based filter (more reliable than checking `attributes.name` per Pitfall 1 in RESEARCH.md).
6. If no matching relations, `console.log(JSON.stringify([]))` and exit 0.
7. For each branch link: call `parseVstfsRefUri(link.url)` then `resolveRepository(org, project, repositoryId, encodedPat)`.
8. Collect results as `[{ repositoryId, repositoryName, remoteUrl, branchName }]`. Skip any repo that fails to resolve (log warning to stderr, continue with others).
9. `console.log(JSON.stringify(results))` and exit 0.
